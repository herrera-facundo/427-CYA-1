## What's New (User Facing)
* Polished user registration experience
* Improved course and file creation views
* Improved dashboard user interface
* Downloadable app release created
* General frontend polish and usability improvements

## Work Summary (Developer Facing)
During Sprint 5, the team focused on preparing the application for deployment and improving the overall user experience. The backend architecture was restructured to better support future features and improve maintainability. Additionally, the frontend was refined to create a more polished and user-friendly interface across core areas such as registration, course creation, and the dashboard.

A major milestone during this sprint was creating a downloadable application release, allowing users to install and run the app more easily. The team also began transitioning toward AWS hosting in preparation for live deployment. These improvements move the application closer to a production-ready state and establish a stronger foundation for future scaling and feature expansion.

## Unfinished Work
Some planned deployment-related work remains incomplete. The migration to AWS hosting has begun but is still in progress. Additionally, the recommendation and progress tracking systems are still under development and will require further implementation in upcoming sprints.

Further testing is also needed for hosting the application and preparing it for deployment on mobile application stores.
